数据流

所有高德 API 调用通过后端代理转发，浏览器不直接调 Amap Web Service：

  地图拖动 → moveend事件
    → amapRegeo(lon,lat) → GET /api/regeo            → Amap 逆地理编码
    → fetchWeather(city)  → GET /api/weather          → Amap 天气查询

  搜索回车 → amapGeocode(q) → GET /api/geocode?address=... → Amap 地理编码
    → map.flyTo(result.lat, result.lon)

需要改动的文件

后端 (Python):
• app/routers/amap.py — 新增 GET /api/regeo 端点

前端 (Vue):
• src/utils/amap.js — amapRegeo 改为调用后端代理
• src/views/MapView.vue — 按新布局全部重写模板+样式

不涉及的代码:
• router、sidebar、header — 不需要改（路由不变）
• useWeather.js — 逻辑不变
• main.py — amap router 已注册，无需再改
